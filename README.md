# AlgoWiz
<p>AlgoWiz is a Python Desktop aap which visualizes different Sorting, Searching and Pathfinding Algorithms.</p>

## Implemented Algorithms:
* Bubble Sort 
* Insertion Sort 
* Selection Sort 
* Merge Sort 
* Quick Sort 
* Heap Sort 
* Counting Sort

## Built With:
* Python 3.7.7
* Tkinter 8.6
